# simple_nodejs_static_file_server

1个简易的nodejs静态文件服务器

## 参数

### host

- 类型: String
- 是否必须：否
- 默认值：`'127.0.0.1'`

### port

- 类型: Number
- 是否必须：否
- 默认值：`3000`

## 使用示例

### 使用默认host和port

```bash
npm start
```

### 使用指定host和port

```bash
npm start 127.0.0.1 3000
```

### 使用默认host和port启动开发环境

```bash
npm run dev
```

### 使用指定host和port启动开发环境

```bash
npm run dev 127.0.0.1 3000
```
